🚀 Cómo instalar y ejecutar

1
Descargar y extraer el ZIP
Descarga el archivo ZIP y extráelo en cualquier carpeta. Ejemplo: C:\PyroShowPro\
2
Opción A — Doble clic (Windows)
Ejecuta INICIAR_SERVIDOR.bat — abre el servidor y el navegador automáticamente.
3
Opción B — Línea de comandos
Abre la terminal en la carpeta del ZIP y ejecuta:
python servidor.py
Luego abre el navegador en: http://localhost:8080/pages/app.html
4
Opción C — Abrir directamente
Abre pages/app.html directamente en el navegador (sin servidor). La app funciona completamente, pero la conexión al ESP32 requiere el servidor local.
5
Para usar con ESP32 WiFi
Conecta tu PC a la red WiFi PyroShow_Master que crea el ESP32. Abre el Panel de Disparo en la app y haz Ping All para verificar.
